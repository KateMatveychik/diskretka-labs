# Программа для поиска наибольшего независимого множества и наименьшего вершинного покрытия

## Описание
Данная программа реализует алгоритм перебора всех возможных независимых множеств вершин графа без использования рекурсии. После нахождения наибольшего независимого множества (MIS) вычисляется наименьшее вершинное покрытие (MVC) как дополнение MIS. Граф представляется списком смежности, а алгоритм использует стек для хранения состояний перебора вершин.

## Функции

### `read_graph_from_file(filename)`
- Читает граф из текстового файла.
- **Аргументы:** `filename` — имя входного файла.
- **Возвращает:** объект класса `Graph`.
- Создаёт список смежности и добавляет рёбра между вершинами.

### `findMIS(graph)`
- Находит наибольшее независимое множество вершин графа.
- **Аргументы:** `graph` — объект класса `Graph`.
- **Возвращает:** множество вершин `MIS`.

### `findMVC_from_MIS(graph, MIS)`
- Находит наименьшее вершинное покрытие графа на основе найденного MIS.
- **Аргументы:** `graph` — объект класса `Graph`, `MIS` — наибольшее независимое множество.
- **Возвращает:** множество вершин `MVC`.

### Класс `Graph`
- **Поля:** `n` — количество вершин, `graph` — список смежности вида `[[], [], ...]`.
- **Методы:** `add_edge(i, j)` — добавляет ребро между вершинами `i` и `j`, `neighbors(v)` — возвращает список соседей вершины `v`, `print_graph()` — печатает список смежности.

## Формат входного файла (`input.txt`)
- Первая строка: число вершин `n`.
- Каждая последующая строка: ребро двумя числами через пробел — индексы вершин.
